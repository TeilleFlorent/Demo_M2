# Demo_M2


Real time demo entirely realise in C++ and openGL-GLSL 3.3.
___________________________________________________________


1) Compile Cmake module :
------------------------- 
cd Objet/ && cmake .. -G "Unix Makefiles" && cd ..

2) Compile Demo : 
-----------------
cd Objet/ && make clean && make && cd ..

3) Run : 
--------
cd bin/ && ./Demo
